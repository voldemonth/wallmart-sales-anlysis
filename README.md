# Walmart Sales Analysis

## Dataset
Walmart Sales Dataset (Kaggle)

# Project Overview
This project analyzes Walmart sales data to identify sales trends, top-performing stores, and the impact of holidays and economic factors on revenue.

# Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

# Key Insights
- Holiday weeks show higher average sales compared to non-holiday weeks.
- Stores 20, 4, 14, 13, and 2 perform best during holiday periods.
- Unemployment and temperature have a weak correlation with weekly sales.

# Conclusion
The analysis highlights that promotional events and store-level factors have a stronger impact on sales than macroeconomic variables.
